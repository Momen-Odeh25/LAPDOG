Our adaptation:\n\n- **Memory Efficient**: Uses Gemma 2B/7B models with quantization\n- **Colab Optimized**: Includes gradient checkpointing, mixed precision, and memory management\n- **Parameter Efficient**: Employs LoRA for fine-tuning only a subset of parameters\n- **User Friendly**: Provides Jupyter notebooks with step-by-step guidance\n\n### Architecture Comparison\n\n| Component | Original LAPDOG | LAPDOG-Gemma (Colab) |\n|-----------|----------------|----------------------|\n| Reader Model | T5-XL/XXL (3B-11B params) | Gemma 2B-7B with LoRA |\n| Retriever | Contriever | Sentence-BERT (lighter) |\n| Memory Usage | ~20-40GB | ~8-12GB |\n| Training | Full fine-tuning | Parameter-efficient |\n| Environment | Multi-GPU clusters | Single Colab GPU |\n\n## üõ†
